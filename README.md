# Water Quality Prediction using Micropollutant Data

## Description
This project is developed as part of the FAIR Data Science course. 
The goal is to predict surface water quality using machine learning techniques based on environmental and chemical data.

## Use Case
We aim to build a classification model that predicts pollution levels in surface water using micropollutant concentrations and environmental parameters such as temperature, pH, and dissolved oxygen.

## Folder Structure
- data/ → input datasets
- src/ → source code
- outputs/ → results, figures, models
- docs/ → documentation files

## File Organisation
- input files: input_<name>.csv
- output files: output_<name>.csv
- scripts: script_<name>.py
- configs: config_<name>.json

## Dataset
The dataset will be sourced from TU Wien Research Data Repository.
